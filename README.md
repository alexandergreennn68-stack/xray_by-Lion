# LionXsec Cloud Tunnel

> Lightweight GitHub Codespaces tunnel based on Xray Core

## Features
- Multi Protocol Ready
- Auto Restart Protection
- Multi Port Support
- Lightweight Debian Container
- Automatic Config Output
- Stable Xray Runtime
- Personalized LionXsec Branding

## Supported Ports
- 443
- 8443
- 2053
- 2083

## Supported Protocols
- VLESS
- VMESS
- Trojan
- Shadowsocks

## Stability Improvements
This edition includes:
- Automatic Xray restart loop
- Better startup handling
- Cleaner runtime scripts
- Dynamic UUID generation
- Multiple inbound ports

## Usage
1. Fork repository
2. Open Codespaces
3. Wait for initialization
4. Your tunnel configs will appear automatically in terminal
